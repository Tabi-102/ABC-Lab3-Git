## Описание
Цель этой лабораторной работы — освоить систему контроля версий **Git**, научиться работать с ветками, слияниями, конфликтами, rebase и управлением историей.

Этот репозиторий создан автоматически на основе персонального варианта **(458789)** через генератор `lab3gen.sh`.

---

## Выполненные задания

### Шаг 1–2: Базовые коммиты  
- Создан файл `notes.txt`
- Добавлены строки `"алгосы"` и `"не алгосы"`

### Шаг 3: Ветка `pr_feature`  
- Создана ветка  
- Добавлена строка `"алгосы"`  
- Сделан коммит

### Шаг 4: Сравнение коммитов  
- Результат сохранён в `diff.txt`

### Шаг 5: Откат к 3-му коммиту  
- Добавлена строка `"алгосы"`  
- Создан новый коммит

### Шаг 6: Rebase ветки `pr_feature` на `main`  
- Конфликты обработаны вручную

### Шаг 7: Merge веток `feature1` и `feature2`  
- Автоматический конфликт  
- Конфликт решён  
- Создан merge commit

### Шаг 8: Ветка `release`  
- Создана ветка  
- Объединены все изменения
